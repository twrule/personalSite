For my Artificial Intellegence course, CSE140, I built a stock market price predictor. This tool used Machine Learning and LSTM units to help learn. When run, the application asks for a stock and generates a csv file with the entirety of that stocks price history. It then trains on all the generated data to predict the stock's closing market price for the following day using indicators based on the Swing Trading model. There is a bit of user customizability such as range of dates to train on, any stock that exists can be used, and the model can predict the next day, week, or month's closing price (next day is the most optimal). The model was adjusted to slightly underpredicts the price to make sure the user would never wait for a stock to hit price that shouldn't happen. Here is a 5 minute presentation on youtube that we did. To build this project we used Python, the Alpha Vantage API, LSTM model, Keras, Plotly and Matlib, and RNN architectures.

Here is the SmartCalendar android application that I built for my Mobile Applications Course, CMPS121, at UCSC. The purpose of the application was to create a "smart calendar" that would allow users to input tasks they have on any given day and the application would help the user better manage their time by giving reccomendations of how long a person should work each day to complete a certain task. If the user did not complete the suggested time, the application would adjust the following days' required time neccessary to complete the application. This application was built using Java, and Android Studio.
